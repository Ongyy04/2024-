##📮시간의 편지
2024 티지톤

##👋🏻 프로젝트 소개
지금까지의 답답했던 기억을 보내고, 특별한 순간을 미래로부터의 선물로 받아보세요. 
시간의 편지는 과거의 자신 또는 친구들에게 보낸 메시지를 미래의 어느 날 받는 웹서비스입니다. 
감성적인 메시지를 오랜 시간 보관했다가 원하는 순간에 받아볼 수 있습니다.

🔗URL http://lettersoftime.com/


##✅ 주요 기능
☝🏻. 소셜 로그인 (카카오)
✌🏻. 미래로 날아가는 편지 쓰기
🤟🏻. 편지 모아보기
